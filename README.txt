1. Install all libraries from requirements.txt
    - I suggest using python virtual environments, see https://docs.python.org/3/library/venv.html
2. Run the server first $python3 chat_server.py -sp [port # you want to run the server on, default is 50005]
3. Run $python3 chat_client.py to start a client application. You can do this as many times as you would like.
    - You will be prompted to either register a new user, log into a registered user, or use one of the preconfigured accounts
        1. alice (password: password1)
        2. bob (password: password2)
        3. charlie (password: password3)
